🔒 Privacidade e Segurança

Pode ficar tranquilo! Não leremos sua carta para a pessoa remetente ou da nossa equipe. Estaremos organizando tudo para que sua mensagem chegue ao destinatário com todo o carinho e cuidado, sem conhecimento prévio de nenhum indivíduo.

A privacidade é nossa prioridade!

🛡️ Segurança do Pagamento

Todos os pagamentos são processados pelo Mercado Pago, uma das plataformas de pagamento mais seguras do Brasil. Seus dados financeiros estão protegidos com criptografia de ponta.

1. Obter chaves de API do Mercado Pago
2. Configurar webhook para confirmação de pagamentos

4. Atualizar a chave pública no JavaScript
